# Setup Authentication

- Configure Firebase sdk: https://firebase.google.com/docs/flutter/setup
- Configure flutter auth: https://firebase.google.com/docs/auth/flutter/start
- Sign in with Apple and Google: https://firebase.google.com/docs/auth/flutter/federated-auth


For google:
1. Make sure the enable URL scheme (see info.plist)
2. URL scheme can be found in https://console.cloud.google.com/auth/clients

For Apple:
1. Follow these steps: https://firebase.google.com/docs/auth/ios/apple
2. Enable Apple Sign in capability in Xcode.
